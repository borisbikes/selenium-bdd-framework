# selenium-bdd-framework
This framework allows you to avoid to start an automation project base on Selenium from scratch.

### Set properties
#### Browser: 
1. chrome
2. firefox
3. edge
4. ie

>eg. -Dbrowser=chrome 
#### Environment:
1. qa
2. staging
>eg.  -Denvironment=qa

### Tools
- Selenium
- Cucumber
- Cucumber Reports
- Apache Logger

### Reporter Result
On this path you can find the test report result
>target/cucumber-html-report.html
